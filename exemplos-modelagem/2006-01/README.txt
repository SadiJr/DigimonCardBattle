ESTRUTURA DE PASTAS DO DIRECTÓRIO:
CodigoFonte: Todos os arquivos .java, .class, imagens e sons do jogo
Documentacao: Especificação de requisitos e todos diagramas UML disponíveis para o jogo
Executaveis: arquivo .jar acompanhado das pastas imagens e media para fácil e rápida execução




Instruções para abrir o código




PARA EXECUTAR O JOGO:
Execute o arquivo DaNumba.jar
OBS: Junto dele, devem estar as pastas imagens/ e media/ para que ele possa encontrar os sons e imagens do jogo.


PARA ABRIR APENAS UM ARQUIVO (.java) EM ESPECÍFICO:
Você poderá utilizar um simples editor de textos (como o bloco de notas do windows e o vi do linux) ou qualquer outra ferramente IDE Java.


PARA ABRIR TODO O PROJETO
1 - Abra no NetBeans 5.0 (não garantimos a compatibilidade com versões mais antigas);
2 - Vá em File > Open Project e selecione a pasta "CodigoFonte" que se encontra neste zip.
3 - Pronto. O projeto e seus arquivos já está aberto.
