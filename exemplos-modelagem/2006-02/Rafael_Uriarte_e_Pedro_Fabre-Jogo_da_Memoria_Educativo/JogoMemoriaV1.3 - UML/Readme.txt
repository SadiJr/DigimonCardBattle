=============================================================
II	Requerimentos Mínimos
=============================================================

Maquina virtual Java;

Um megabyte de espaço livre em disco;

O jogo foi criado e testado com a versão 1.6 da maquina virtual
java, assim recomendamos o uso da mesma.

=============================================================
III	Instalação/Utilização
=============================================================

-Para instalação do jogo, descompacte o arquivo .zip em um di-
retorio e rode o arquivo JogoMemoria.jar

-Para alteraração dos componentes do jogo renomeio os novos
componentes usando o padrão atual dos diretorios dos diretorios.
Os textos, nomes e dicas devem ser em texto puro e a imagem deve
ser no formato JPG(retirando as extenções).

=============================================================
IV	Conteúdo da pasta descompactada
=============================================================

Na descompactação será gerado o seguinte conteúdo:

/JogoMemoriaV1.3/			              Diretório principal

/JogoMemoriaV1.3/Readme.txt		              Este arquivo de texto

/JogoMemoriaV1.3/JogoMemoria.jar        	      Executavel Jar do jogo

/JogoMemoriaV1.3/doc/			              Diretório da documentação

/JogoMemoriaV1.3/doc/UML - Jogo da Memoria Educativa.jude Arquivo JUDE contendo a especificação UML

/JogoMemoriaV1.3/doc/Auto-Avaliação.rtf	              Documento contendo uma avaliação sobre o projeto em geral.

/JogoMemoriaV1.3/doc/Requisitos-Memoria-Educativa-1.3 Especificação de requisitos do Software.

/JogoMemoriaV1.3/src/			              Diretório principal do código fonte do software

/JogoMemoriaV1.3/dicas/                               Diretório contendo as dicas para procura do par correto

/JogoMemoriaV1.3/imagems/			      Diretório contendo as imagens das cartas

/JogoMemoriaV1.3/nomes/				      Diretório contendo os nomes das cartas 	

/JogoMemoriaV1.3/textos/			      Diretório contendo os textos dos pares corretos

/JogoMemoriaV1.3/sistema/			      Diretório que contem os arquivos de sistema para persistencia	

/JogoMemoriaV1.3/sistema/FiguraPadrao/                Figuras usadas pelo Jogo.
